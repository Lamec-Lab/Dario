REFERENCE TYPES =================================================================
p.s.: - the fields with {*} are mandatory 
      - fields that can be replaced, must have at least one of 
        the options if it is mandatory, but can have all then.
      - Prefix as "dos", "das", "von" need to be placed as {das}
      - these references are the standards by ISO8859-15.
      - accents and symbols must follow the LaTeX coding as:
            - \`a = à
            - \´a = á
            - \~a = ã
            - \^a = â
            - \c c = ç
